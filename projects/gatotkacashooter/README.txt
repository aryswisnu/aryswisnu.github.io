=============================Gatotkaca Shooter====================
Web based:
1. Klik 2x pada "index.html" (berjalan baik pada browser Google Chrome).
2. Gerakkan gatotkaca menggunakan button arrow.
3. Tombol space untuk menembak.

Android:
1. File android berada pada folder "apk" dengan nama file "Gatotkaca Shooter".
2. Pindahkan ke perangkat android
3. Install pada perangkat android
4. Jika instalasi berhasil, tekan "Open" atau "Buka".
5. Arahkan dengan menggeser jari yang menyentuh layar ke kiri-kanan.
